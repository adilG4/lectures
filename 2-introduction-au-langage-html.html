Les balises "en paires"

Elle fonctionnent par deux : une balise ouvrante et une balise fermante. La balise fermante se distingue par l'ajout du caractre "/".

Une balise de ce type agit sur le texte qu'elle encadre.

Exemples:

  • <h1> CE TEXTE EST UN TITRE </h1>
  • <b> Ce texte est en gras </b>
  • <p> Ce texte est un paragraphe </p>

Imbrication de balises

Les balises HTML s'imbriquent. Exemple :

  • <p> Du texte <b> en gras </b> dans un paragraphe. </p>

La proprit CSS display

Ce mode d'affichage peut tre modifi par la proprit CSS display.

Chaque balise a donc un display par dfaut, et la plupart peuvent se classer dans une des deux catgories block ou inline.

block et inline sont les deux principaux display mais il existe d'autres types de display : inline-block, table, table-cell, ...
