Introduction

L'objectif des tableaux est de prsenter des donnes tabulaires structures en lignes et colonnes, comme dans un tableur tel qu'OpenOffice Calc ou Microsoft Excel.

Les balises relatives aux tableaux

  • <table> : Dfinit un tableau
  • <caption> : Encadre le titre du tableau
  • <tr> : Cre une nouvelle ligne dans un tableau (tr = table row)
  • <th> : Cre une cellule d'entte dans une ligne de tableau (th = table header)
  • <td> : Cre une cellule classique dans une ligne de tableau (td = table data)

Fusionner des colonnes

L'attribut colspan permet de fusionner plusieurs colonnes au niveau d'une ligne. La valeur donne correspond au nombre de colonnes fusionner.

Exemple

<table>
  <tr>
    <td>A</td>
    <td>B</td>
    <td>C</td>
  </tr>
  <tr>
    <td colspan="2">1</td>
    <td>2</td>
  </tr>
</table>

Fusionner des lignes

L'attribut rowspan permet de fusionner plusieurs lignes au niveau d'une colonne. La valeur donne correspond au nombre de lignes fusionner.

Exemple

<table class="extable">
  <tr>
    <td rowspan="2">1</td>
    <td>A</td>
  </tr>
  <tr>
    <td>B</td>
  </tr>
  <tr>
    <td>2</td>
    <td>C</td>
  </tr>
</table>

Utilisation des tableaux

Quand utiliser un tableau

Il faut penser la smantique du code HTML : <table> = tableau !

On utilise donc un tableau pour afficher des donnes tabulaires. Exemple :

  • Tableau de prix
  • Statistiques
  • ...

Quand ne pas utiliser un tableau

On n'utilise pas un tableau pour faire de la mise en page. C'est une mthode obsolte bannir.

  • Ce n'est pas responsive
  • On ne peut pas modifier la mise en page avec du CSS

Retour de la proprit display

La proprit display permet de simuler l'affichage en tableau grce plusieurs valeur : table, table-caption, table-cell, ...

  • display: table : permet un lment de se comporter comme un tableau
  • display: table-row : permet un lment de se comporter comme une ligne de tableau
  • display: table-cell : permet un lment de se comporter comme une cellule de tableau
  • ...

Avantages

Par rapport la mthode obsolte avec table

  • Le code HTML reste simple et souple
  • On peut changer la mise en page avec du CSS, par exemple :
    • passer le menu droite plutt qu' gauche
    • transformer le menu vertical en menu horizontal au dessus du contenu
    • ...
  • On ne va pas l'encontre de la smantique des balises HTML

Par rapport l'utilisation de float

  • Les lments ne sortent pas du flux HTML
    • Pas de problme de marges
    • Pas de problme de conteneurs vides
